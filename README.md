## Dicionário Git do QA

| Termo | Significado |
|---|---|
| Commit | Salvar alterações realizadas com uma mensagem descritiva. |
| Push | Enviar os commits do repositório local para o GitHub |
| Pull | Atualizar o repositório local com alterações do repositório remoto |
| Merge | Unir alterações de diferentes branches em uma única branch. |
| Branch | Uma ramificação do projeto usada para desenvolver funcionalidades sem alterar a principal. |
